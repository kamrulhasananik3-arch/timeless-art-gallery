# Timeless Art Gallery

Simple static gallery demo showcasing artworks and upload UI.

Preview locally:

- Open `index.html` in your browser (double-click or use Live Server).

Notes before publishing to GitHub:

- This is a static front-end only. The upload page simulates uploads in-browser; no backend is included.
- Ensure you have rights to any images you publish (some images here are external examples).

Optional next steps:

- Add a LICENSE file (e.g., MIT) if you want to open-source.
- Add a small server/backend for handling real uploads.
